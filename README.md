
# NovaCiv — Clean Foundation

Это чистый каркас сайта (Vite + React + Tailwind) с простой мультиязычностью и загрузкой текстов из `/public/txt`.
Он создан, чтобы без боли разместить финальные тексты Манифеста и Устава **в точности** как в утвержденных .txt-файлах.

## Структура
- `public/txt/Manifest-*.txt` — финальные тексты Манифеста.
- `public/txt/Charter-*.txt`  — финальные тексты Устава.
- Языки: `ru`, `en`, `de`, `es` (можно добавить другие по аналогии).

## Как запустить
1) Установите зависимости:
   ```
   npm i
   ```
2) Локальный запуск:
   ```
   npm run dev
   ```
3) Сборка и предпросмотр:
   ```
   npm run build && npm run preview
   ```

## Как опубликовать на Netlify
- Репозиторий → Netlify → Build command: `npm run build`
- Publish directory: `dist`

## Как добавить тексты
- Скопируйте **утвержденные** файлы Манифеста/Устава в `public/txt/` строго под именами:
  - `Manifest-ru.txt`, `Manifest-en.txt`, `Manifest-de.txt`, `Manifest-es.txt`
  - `Charter-ru.txt`, `Charter-en.txt`, `Charter-de.txt`, `Charter-es.txt`
- Пересборка **не требуется** — фронтенд читает файлы из `/public` на лету.
- Обновите страницу (shift+refresh), чтобы увидеть изменения.

## Примечание
- Никаких "заглушек" в тексте: если файл пустой или отсутствует, страница честно покажет блок «Текст не найден».
  Это сделано, чтобы **никогда** не подменять официальный текст черновиком.
